*****PROJETO APPSTORM*****

Seguir seguintes passos para funcionamento correto do programa:

1- Desempacotar o arquivo .zip
2- Abra o Xampp e inicialize o "Mysql"
3- Clique no arquivo "projetoappstorm" e pressione o botao shift + mouse2
4- Clique na opção "Abrir com Windows PowerShell"
5- Rode o seguinte comando: php artisan migrate
6- Em seguida: php artisan db:seed
7- E por último: php artisan serve
8- No navegador, vá para localhost:8000/produtos

Obs.: O programa não está 100% terminado como pode se observar porém, como um dos diferenciais 
era o framework Laravel, optei por tentar utilizá-lo e também aprender um pouco mais sobre
