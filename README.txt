*****PROJETO APPSTORM*****

Seguir seguintes passos para funcionamento correto do programa:

1- Abra o Xampp e inicialize o "Mysql"
2- Crie uma pasta "projetoappstorm" e insira todos os arquivos deste repositório, e entao pressione shift + mouse2
3- Clique na opção "Abrir com Windows PowerShell"
4- Rode o seguinte comando: php artisan migrate
5- Em seguida: php artisan db:seed
6- E por último: php artisan serve
7- No navegador, vá para localhost:8000/produtos

Obs.: O programa não está 100% terminado como pode se observar porém, como um dos diferenciais
